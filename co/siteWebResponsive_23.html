Mise en Application _ Pas à Pas

Pen de départ

http://codepen.io/ejallier/pen/PwJQMM

Cette page a des colonnes fluides.

Nous allons écrire les media queries nécessaires pour que sur un petit écran, les 2 colonnes se retrouvent l'une après l'autre.

Préalable dans le head

Il faut rajouter ceci dans le head pour que les media-queries fonctionnent sur smartPhone

<meta name="viewport" content="width=device-width">

Dans content, vous pouez rajouter d'autres infos, (séparées par des virgules) :

  • user-scalable=no (pour empécher de zoomer/dézoomer)

  • initial-scale contrôle le niveau de zoom lorsque la page est chargée pour la première fois

  • maximum-scale, minimum-scale, contrôlent la manière dont les utilisateurs sont autorisés à zoomer ou dézoomer une page.

https://developer.mozilla.org/fr/docs/Mozilla/Mobile/Balise_meta_viewport#Les_bases_du.C2.A0viewport

Media Querie

Après quelques test, il apparaît qu'un point de rupture peut être placé en dessous de 700px :

@media screen and (max-device-width : 700px){

}

A cet endroit, je vais pouvoir modifier mon css :

@media screen and (max-device-width : 700px){

  .col1, .col2{

    float : none; /* on annule le float */

    width: auto; /* on rétabli la taille normale */

  }

}
