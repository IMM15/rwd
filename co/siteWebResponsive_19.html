Les grilles du futur : le modèle flex

Les colonnes avec Flex

la méthode flex devrait devenir LA méthode pour définir des colonnes dans une page web.

Ecrivez dans le conteneur des colonnes :

.flexbox-container {

display: -ms-flex;

display: -webkit-flex;

display: flex;

}

Et voilà !

Plus besoin de clearfix, de détourner les float ou les inline-block de leur usage premier.

On peut dire que display : flex ; est la VRAI méthode pour faire des colonnes en CSS, celle qui manquait jusqu'à présent, que tous les dev CSS attendaient depuis des temps immémoriaux.

La méthode

http://davidwalsh.name/flexbox-column

Sur le main :

main{

margin-bottom: 3em;

display: -ms-flex;

display: -webkit-flex;

display: flex;

}

Pour les colonnes, il suffit de définir les largeurs, comme précédemment... Ou presque :

La marge est prise à part, les pourcentages se répartissent dans l'espace disponible, marges exclues.

On peut donc se servir du principe des colonnes comme pour bootstrap

.col1{

background: #ccc;

width: calc(100%/12*3);

padding: 1em;

margin-right: 1em;

}

.col2{

background: #eee;

width:calc(100%/12*9) ;

padding: 1em;

}

Le margin-right du .col1 peut être modifié, sans que l'on soit obligé de modifier les width.
